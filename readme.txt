== Description ==

**SEO Stats Widget** Plugin collects SEO statistic from various sources like Google, SemRush, Alexa, Dmoz and shows all stats in admin dashboard and sidebar widget.

= Currently this plugin shows following stats =

* Google Pagerank
* Google Indexed Pages
* Google Backlinks
* Keywords in Google
* Keywords Cost (PPC)
* Organic Traffic (From Google)
* SemRush Rank
* Alexa Rank 
* Alexa Backlink 
* Dmoz Listing 
* Website Load time

**SEO Stats Widget**  by [99webtools](http://99webtools.com "Free Web Tools")

== Installation ==

= Using the Plugin Manager =

1. Click Plugins
2. Click Add New
3. Search for `seo-stats-widget`
4. Click Install
5. Click Install Now
6. Click Activate Plugin

= Manually =

1. Upload `seo-stats-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
